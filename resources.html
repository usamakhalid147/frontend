Additional Data Availability

All the computational scripts used in this work (Script Files 1–12) are available through GitHub at the Functional Genetics Laboratory page, under the repository named ‘NB-lncRNAs’: https://github.com/FunctionalGeneticsLab/NB-lncRNAs and https://doi.org/10.5281/zenodo.7844207. Scripts for bulk RNAseq analysis used for tissue-specificity analyses are available at https://github.com/FunctionalGeneticsLab/GRADE and https://doi.org/10.5281/zenodo.7844215.
